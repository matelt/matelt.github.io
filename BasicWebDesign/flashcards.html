Interactive Flashcards

Transitions
CSS property that allows changes to CSS property values to occur smoothly over a specified duration.
Animations
CSS property that enables the animation of HTML elements by gradually changing their properties through keyframes.
rotate3d
A CSS function that rotates an element around a specified 3D vector (x, y, z) by a given angle.
translate3d
A CSS function that moves an element in 3D space along the x, y, and z axes.
transform-style
A CSS property that specifies how nested elements are rendered in 3D space.
@keyframes
A CSS at-rule used to define the intermediate steps of an animation sequence by specifying keyframes.
Vanishing Point
In graphics, the point where parallel lines appear to converge in 3D space, giving the illusion of depth.
perspective-origin
A CSS property that defines the position from which the viewer is looking at the 3D-transformed elements.
